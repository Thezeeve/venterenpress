# Public Discovery Layer

The public newsroom now exposes the main editorial entry points through a shared navigation shell and dedicated landing pages.

## Navigation

- Desktop mega menu groups categories, regions, live coverage, video, opinion, newsletters, and subscription links.
- Mobile users get a drawer with the same route set plus edition switching.
- Breaking banners are surfaced in the sticky header and reused by the homepage ticker.

## Landing Pages

- `/latest` for the chronological newsroom feed.
- `/most-read` for audience-ranked stories.
- `/topics/[slug]` for topic-style pages that resolve to categories or tags.
- `/categories/[slug]` for desk pages.
- `/regions/[slug]` for edition and region pages.
- `/tags/[slug]` for tag-based browsing.
- `/rss` for the feed directory.

## Reader Trust

- Article pages show editorial review status, source transparency, correction history, and a "Why this story matters" card.
- Logged-in readers see personalized continue-reading and recommendations derived from bookmarks, follows, editions, and reading history.
- Notification preferences and newsletter controls live in the account area and are surfaced in the header dropdown.
